# UX Observer

Detects UX failures. If detected → trigger refinement (designer + simplifier).

## Detect

- Unclear primary action
- Too many choices on one screen
- Weak hierarchy (everything same weight)
- Confusing flow (RFQ funnel broken)
- Copy too generic or vague
- Section without clear purpose

## Everest-specific

- Duplicate CTAs outside Hero/form
- Russia→CN narrative emphasis
- Missing i18n on visible strings

## Action

Log issue → invoke simplifier → update plan → do not ship until clarity restored.
