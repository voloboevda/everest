# Color System

Use color sparingly. Typography and spacing create hierarchy before color.

## Principles

Minimal · premium · accessible · high contrast.

80% neutral · 15% accent · 5% semantic.

Never rainbow palettes. Never more than one accent color competing per section.

## Everest brand (fixed — do not invent)

| Role | Hex | Usage |
|------|-----|-------|
| Background | `#0A0A0B` | Page, sections |
| Logo navy | `#003048` | Brand reference |
| Accent blue | `#3D7F96` | Links, interactive, odd title accents |
| Gold | `#C8A45A` | Even title accents, sparing highlights |
| Text primary | `rgba(232,236,242,0.92)` | Headlines |
| Text muted | `rgba(232,236,242,0.62)` | Body secondary |

Alternate rhythm: odd sections blue accent · even sections gold accent (block eyebrows).

## Usage rules

| Category | Rule |
|----------|------|
| Primary | Logo-family blue for interaction |
| Secondary | Gold accent only — not mixed randomly with blue in same element |
| Neutral | Most of interface |
| Success / Warning / Error | Semantic only (forms, validation) |

## Backgrounds

Prefer: near-black · neutral gray · subtle dark surfaces.

Avoid: saturated backgrounds · noisy gradients · multiple competing surfaces.

## Premium feel

Comes from typography · spacing · alignment · consistency — not bright colors.

## Contrast

Meet accessibility requirements. Text readable in all states. Never sacrifice readability for aesthetics.

## When choosing colors

1. Start with neutrals
2. Use one accent direction (blue OR gold emphasis per element)
3. Semantic colors only for status
4. Never add colors without reason

## Reference palettes (inspiration only)

- Linear: near-black + violet accent
- Vercel: black/white, minimal accent
- Stripe: blue + subtle purple
- Raycast: dark + soft blue
