# China review checklist

Before showing site to Chinese partners / China agent group.

## Content

- [ ] 中文 reads native B2B, not machine-translated tone
- [ ] Trade terminology correct (FOB/CIF/CPT, 大宗贸易, 供应链)
- [ ] Partner tone — 合作/partnership, not sales hype
- [ ] No politically sensitive map labels (verify geography block)

## UX

- [ ] Noto Sans SC renders correctly on mobile
- [ ] WeChat button visible and works (QR or placeholder noted)
- [ ] Page loads acceptably from China (test via CN proxy if possible)
- [ ] No broken Google-dependent assets on CHI version

## Analytics

- [ ] Baidu Tongji snippet on CHI version (or stub documented)
- [ ] Do not rely on GA4 for CN traffic reports
